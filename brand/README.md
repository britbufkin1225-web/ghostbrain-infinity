# GhostBrain Infinity — Brand Identity Lock

## Project Identity

Project Name: GhostBrain Infinity  
Parent Brand: devdevbuilds  
Product Type: Local-first AI / knowledge / source-intelligence dashboard  

## Brand Hierarchy

devdevbuilds
└── GhostBrain Infinity

GhostBrain Infinity is the primary product identity.  
devdevbuilds is the parent studio/build brand and should appear only as a smaller supporting credit.

## Visual Direction

GhostBrain Infinity should feel:

- dark metallic
- premium
- futuristic
- graph-driven
- local-first
- AI-system oriented
- Obsidian-inspired
- chrome/platinum over black glass

Avoid:

- childish mascot styling
- random neon colors
- flat generic SaaS styling
- oversized distracting brand art
- reused broken-dashboard styling

Primary UI: Inter, Geist, or system sans-serif

## Dashboard Placement Rules

- GhostBrain Infinity logo should appear in the primary header area.
- devdevbuilds should appear as a smaller studio credit.
- The ghost/brain icon may be used as a subtle background mark.
- Background mark opacity should stay between 0.04 and 0.12.
- Brand imagery must never block graph nodes, controls, inspectors, registry panels, or dashboard text.
- The brand should support the dashboard logic, not overpower it.
