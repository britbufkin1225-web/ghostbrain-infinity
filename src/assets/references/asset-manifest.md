## Phase 2 Integration Notes

Prefer transparent assets in the live dashboard. Opaque assets are references and preview-safe fallbacks.

The GhostBrain Infinity mark should remain the dominant project identity. devdevbuilds is always a secondary studio credit.

Use the watermark only behind quiet graph areas and never above text, controls, inspectors, registry panels, or graph nodes.
